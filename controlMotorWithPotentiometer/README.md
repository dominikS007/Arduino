# Motor with potentiometer
Changing the potentiometer changes the RPM of the motor 

### Components needed 
* Arduino Uno R3
* DC Motor
* 9V Battery
* 250 kΩ, Potentiometer
* Pushbutton (x2)
* 10 kΩ Resistor (x2)
* H-bridge Motor Driver
